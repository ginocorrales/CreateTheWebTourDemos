Auto-sizing regions

CSS Regions auto-sizing

This sample tests CSS Regions auto-sizing as per the May 2012 Working Draft.

Given the styling of the regions in this page, three regions should be visible, with red, green and blue borders and the layout should look as follows.

The top (red border) region should be as wide as the page. It should only be as high as to contain the first heading and the blockquote about quotes (since the content should break before the second heading).

The left (green border) region should start with the second heading and it should be wider than it is tall (since the max-height is about half the width for this region).

The right (blue border) region should be as half as wide as the left region but tall enough to accomodate for the whole remainder of the text. Another heading with some text after it should be visible.

If the page is wide enough, this text should start flowing from between the left and right regions and then continue after them.
