Styling content when it flows into a region

Tip: resize the browser window so that content flows into the second region and turns red.

Content can be styled differently based on the region it flows into by using the @region rule. For example, the content that flows from this region to the next one will change color from black to red.

Code sample

@region #region2{
    p{
        color: red;
    }
}

Paragraph content which flows into #region2 will have red font color.

When working with named flows and regions the original cascade of styles is not affected. This means that the original styles of the content get applied. Styles defined on the region elements do not affect the styles of the content which flows into them.

The styles of the content which flows into a region can be altered by defining them under a @region rule. This works in a similar way to the @media rule for media queries.
